# Infrastructure Automation

Collection of automation scripts and operational tools focused on enterprise IT infrastructure.

## Planned Projects

### Windows Server Health Check

Automated validation of:

* CPU
* Memory
* Disk Capacity
* Service Status

### Active Directory User Report

Generate reports for:

* Inactive Users
* Expired Passwords
* Disabled Accounts

### Azure VM Inventory

Collect information about:

* Virtual Machines
* Resource Groups
* Subscription Usage

### Microsoft 365 Reporting

Generate reports for:

* License Usage
* Mailbox Statistics
* Security Compliance

## Roadmap

* PowerShell Automation
* Azure Automation
